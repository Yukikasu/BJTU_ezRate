# 🎓 知行乐评

![Version](https://img.shields.io/badge/version-1.0.1-blue.svg)
![Tampermonkey](https://img.shields.io/badge/Tampermonkey-Required-success.svg)
![Platform](https://img.shields.io/badge/Platform-BJTU%20%7C%20ICourse163-lightgrey.svg)

**一键式满分评分脚本，告别手误与重复劳动，节省宝贵时间！**

本脚本是一个基于 Tampermonkey（油猴）的浏览器用户脚本，专为**北京交通大学教务系统（MIS）评教**与**中国大学 MOOC（慕课）互评**设计.

---

## ✨ 功能特点

- 🚀 **双平台支持**：同时支持北京交通大学教务系统评教与慕课（ICourse163）学生互评。
- 🖱️ **便捷悬浮窗**：页面右侧自动加载半透明的“知行乐评”操作面板，支持鼠标拖拽，不遮挡页面主内容。
- 💯 **一键满分评教（BJTU）**：自动识别并勾选“非常符合”或“优秀”选项，自动填写好评评语，并自动滚动至页面底部等待提交。
- 🔄 **批量自动互评（慕课）**：支持设置“互评份数”，自动勾选最高分、填写好评评语、自动点击提交并进入下一份，全程自动化处理。

---

## 🛠️ 安装指南

1. **安装 Tampermonkey 插件**
   - 如果您的浏览器尚未安装 Tampermonkey 插件，请先前往 [Tampermonkey 官网](https://www.tampermonkey.net/) 下载并安装适用于您浏览器的版本（支持 Chrome, Edge, Firefox 等）。
2. **安装本脚本**
   - 方式一：在 Tampermonkey 控制面板中点击“添加新脚本”，将本项目中的代码复制并粘贴进去，按 `Ctrl + S` 保存。
   - 方式二：如果已发布至 GreasyFork，直接点击安装链接（此处可留空待替换您的脚本发布地址）。

## 📖 使用说明

脚本安装并启用后，当您进入支持的页面时，页面右侧会自动出现一个蓝色的【知行乐评】悬浮窗。

### 场景一：北京交通大学教务系统评教
**网址匹配：** `https://aa.bjtu.edu.cn/teaching_assessment/stu/*`

1. 登录教务系统并进入对应课程的教师评教页面。
2. 页面加载完成后，点击悬浮窗中的 **“一键好评”** 按钮。
3. 脚本将自动完成以下操作：
   - 扫描当前页面的所有评价指标，自动勾选“非常符合”或“优秀”。
   - 在主观评价输入框中自动填写：“*老师讲得很好,让我收获良多！*”。
   - 页面自动滚动至最底部。
4. 确认无误后，手动点击页面自带的“提交”按钮即可。

### 场景二：中国大学 MOOC (ICourse163) 学生互评
**网址匹配：** `https://www.icourse163.org/*`

1. 登录慕课并进入作业的“学生互评”页面。
2. 在悬浮窗的 **“互评份数”** 输入框中填写您想要自动批改的作业数量（例如：`5`）。
   - *提示：如果不填数字，点击按钮则只批改当前这一份。*
3. 点击 **“一键好评”** 按钮。
4. 脚本将自动开始循环执行以下操作：
   - 自动勾选每一项评分的最后一个选项（通常是满分）。
   - 在评语框中自动填写：“*写得很好，继续努力！*”。
   - 自动点击提交并翻到下一份作业。
   - 每次提交后会暂停 1.3 秒以等待页面刷新，直到完成您设置的份数。

## ⚠️ 注意事项与进阶配置

1. **慕课并发限制**：在慕课批量互评时，如果网络较慢或遇到系统提示“操作过于频繁/并发限制”，您可以手动修改脚本代码中的延迟时间。
   - 打开脚本编辑器，找到第 95 行 `await new Promise(resolve => setTimeout(resolve, 1300));`，将 `1300`（即1.3秒）适当调大，例如改为 `2000`（2秒）。
2. **拖拽功能**：如果悬浮窗挡住了重要内容，可以按住悬浮窗底部的半透明条带（拖拽区）进行自由移动。
3. **免责声明**：本脚本仅供学习交流与节省重复点击时间使用。自动提交的评语为固定模板，请根据实际情况谨慎使用，避免因机器盲目打分导致的作业评估不准确问题。使用本脚本产生的任何后果由使用者自行承担。
